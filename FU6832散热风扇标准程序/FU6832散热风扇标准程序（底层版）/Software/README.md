# Standard Software for Cooling Fan

> 这是基于FU6832的无感FOC落地扇标准程序  
> 如有BUG或者优化建议，可提交到标准程序对应的议题

## 适用编译器


- [支持] KeilC51
- [不支持] FTCC

## 适用芯片
- FU6832
